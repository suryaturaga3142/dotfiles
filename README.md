# dotfiles
My dotfiles and configs for CachyOS Hyprland. I keep things pretty simple and minimal, nothing fancy. My main priority has been keeping a unified workflow across vim, tmux and hyprland. That was super important to me.

I think I've done a good job at the unified workflow aspect. At least, I've come to the point where it's all in my muscle memory now, and I don't really work with any other setup. As we all know, though, this is a constantly evolving repo. We'll see what changes come my way, especially since I'm using Hyprland lol. My scripting has gone up significantly since I switched.

I'm proud of my waybar, it's sleek and not distracting. Gives me all the info I need and doesn't clutter the bar. Took me a couple days to get right.

Hope you like it! I'll be sure to upload some good pictures of my desktop soon.
